# Project Process Photos Guide

Add your photos to each project's process folder:

## Types of Photos to Add:

1. **Process Photos** (`/process/` folder):
   - Wireframes and sketches
   - Whiteboard sessions
   - User journey maps
   - Mood boards
   - Design iterations

2. **Design Photos** (root folder):
   - Figma/Adobe designs
   - Prototypes
   - Design system components

3. **Development Photos** (root folder):
   - Code screenshots
   - Architecture diagrams
   - Database schemas

4. **Result Photos** (root folder):
   - Final implementations
   - Client meetings
   - Launch events
   - User testing sessions

## Naming Convention:
- process-1.webp, process-2.webp
- design-1.webp, design-2.webp
- code-1.webp, code-2.webp
- result-1.webp, result-2.webp

## Image Specifications:
- Format: WebP (85% quality)
- Size: 1200x800px minimum
- Aspect Ratio: 4:3 or 16:9
